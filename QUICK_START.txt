📋 启动步骤
-----------
1. 打开命令行，进入项目目录:
   cd "d:\StudyCode\AI项目\demo7"

2. 启动系统:
   run.bat

3. 等待启动完成后访问:
   http://localhost:3000

🎯 功能预览
-----------
✓ 智能对话 - 基于阿里千问API
✓ PDF知识库 - 文档上传和解析
✓ 智能检索 - 知识库优先搜索
✓ 现代界面 - 类ChatGPT设计

🛠️ 其他启动方式
---------------
# 开发模式（推荐）
run.bat

# 生产模式（后台运行）
run.bat prod

# 仅启动后端
run.bat backend

# 仅启动前端
run.bat frontend

# 手动启动
cd backend && python app.py
cd frontend && npm start

💡 使用提示
-----------
1. 首次使用建议上传PDF文档构建知识库
2. 支持拖拽上传，最大16MB
3. 对话会自动判断从知识库还是API获取答案
4. 可以在右侧面板管理已上传文档
